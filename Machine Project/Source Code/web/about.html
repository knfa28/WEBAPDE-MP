About

Simon Says is an online application
which keeps track of your daily activities,
tasks, events, appointments, etc.
in an organized manner.

With Simon Says, you will be able to stay up to date
as reminders and alerts will be sent to your devices
via email notifications for as long as you have
registered your account.

Say goodbye to missed deadlines or late appointments
as you can set the priorities of each task you create
as well as select when or how often you would like to receive
reminders prior to the assigned due date.
